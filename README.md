# GitHub 文件分享平台

一个现代化的 GitHub 主页，支持文件上传和管理功能。

## 功能特性

- 🚀 现代化的响应式设计
- 📁 拖拽上传文件
- 📱 移动端友好
- 💾 本地存储文件信息
- 🎨 优雅的动画效果
- 📊 文件类型识别

## 技术栈

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Font Awesome 图标

## 使用方法

1. 克隆或下载此仓库
2. 在浏览器中打开 `index.html`
3. 开始上传和管理文件

## 文件结构

```
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # 功能脚本
└── README.md       # 说明文档
```

## 功能说明

### 文件上传
- 支持拖拽上传
- 支持点击选择文件
- 文件大小限制：10MB
- 实时上传进度显示

### 文件管理
- 文件列表展示
- 文件类型图标识别
- 下载和删除功能
- 本地存储持久化

### 响应式设计
- 桌面端优化显示
- 平板端适配
- 移动端友好界面

## 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择源分支（通常是 main 或 master）
4. 访问 `https://username.github.io/repository-name`

## 自定义

你可以轻松自定义：
- 颜色主题（修改 CSS 变量）
- 文件大小限制（修改 JavaScript）
- 支持的文件类型
- 界面文案和语言
